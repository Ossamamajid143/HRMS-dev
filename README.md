# HRMS backend

Corporate Human Resource Management System backend built with **TypeScript**, **Express**, and **Drizzle ORM**.

## Features
- **Automatic Admin Seeding**: System initializes with an admin account on startup.
- **Attendance System**: Automatic server-time recording for check-in and check-out.
- **Role-Based Access**: Secure endpoints restricted to Admin/HR roles.
- **Type-Safe Validation**: Integrated with Zod for request body verification.
- **API Documentation**: Interactive Swagger UI.
